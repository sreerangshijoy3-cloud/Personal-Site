# Sreerang Shijoy | Personal Portfolio

Welcome to my personal portfolio website! This site is designed to showcase my profile, skills, projects, and interests as I embark on my journey to become a skilled developer.

## About the Project
This project is a modern, responsive personal website built from scratch using:
- **HTML5**: For semantic structure.
- **CSS3**: For styling, layout (Flexbox/Grid), and responsive design.
- **JavaScript**: For interactive features.

## Features
- **Responsive Layout**: Adapts automatically to mobile, tablet, and desktop screens.
- **Navigation**: Easy-to-use sticky navigation bar for smooth browsing.
- **Sectioned Content**: Clearly organized sections for About, Skills, Projects, and Contact information.
- **Modern UI**: Clean design with subtle animations and hover effects.

## How to Run
1. Download or clone this repository to your local machine.
2. Locate the `index.html` file.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.) to view your portfolio.

## Structure
- `index.html`: Contains all the HTML structure, CSS styles (in the `<style>` tag), and JavaScript (in the `<script>` tag).
